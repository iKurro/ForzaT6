# ForzaT6

App web personal para crear builds y reglajes de coches en Forza Horizon 6.

## Objetivo

La app permitirá:

- elegir coche
- elegir clase objetivo: D, C, B, A, S1, S2, R, X.
- elegir tipo de uso: carretera, rally, offroad, drift o drag
- generar piezas recomendadas
- generar reglajes base
- guardar builds favoritas
- añadir notas personales de prueba

## Tecnología prevista

- React
- Vite
- Tailwind CSS
- LocalStorage para guardar datos al principio
